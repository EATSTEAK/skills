# Skills

This repository is a Vercel skills-compatible skill repo.

## Install

Install all skills from this repository:

```bash
npx skills add <owner/repo>
```

Install a specific skill:

```bash
npx skills add <owner/repo> --skill research-first
```

## Available skills

### research-first

Start with deep research before planning. Use when detailed investigation, comparison, or validation is needed before implementation.

Path: `skills/research-first/SKILL.md`

### use-uv

Always use uv for Python work. Use when Python code, dependencies, virtual environments, scripts, or Python CLI tools are involved.

Path: `skills/use-uv/SKILL.md`

### orchestrator

Orchestrate a multi-stage refactor or migration by creating a fresh team per stage, running one implementation subagent and one validation subagent, then cleaning up before the next stage.

Path: `skills/orchestrator/SKILL.md`

### commit

Analyze staged git changes and create one or more well-scoped commits.

Path: `skills/commit/SKILL.md`

### pr

Prepare and create a GitHub pull request from the current branch.

Path: `skills/pr/SKILL.md`

### review

Perform a code review for a branch diff, staged diff, or specific file.

Path: `skills/review/SKILL.md`

### summary

Summarize current branch changes against a base branch.

Path: `skills/summary/SKILL.md`

### rearrange

Analyze and reorganize current branch commits for easier review.

Path: `skills/rearrange/SKILL.md`

### pdftomd

Convert PDF files to Markdown while preserving content accuracy and document structure.

Path: `skills/pdftomd/SKILL.md`
